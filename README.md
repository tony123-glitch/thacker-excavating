## Estimate Form Text Setup (Twilio)

The estimate form posts to `POST /api/estimate` and sends a text message through Twilio.

Create a `.env.local` file with:

```bash
TWILIO_ACCOUNT_SID=your_twilio_account_sid
TWILIO_AUTH_TOKEN=your_twilio_auth_token
TWILIO_FROM_PHONE=+12175550123
ESTIMATE_TO_PHONE=+12175559876
```

Notes:

- Phone numbers must be in E.164 format (example: `+12175550123`).
- `TWILIO_FROM_PHONE` must be a Twilio number that can send SMS.
- If env vars are missing, the API returns a configuration error instead of silently failing.
